# PressArk Website UI Kit — Elevated

A **1→100 upgrade** of the PressArk marketing site. Preserves brand DNA (Inter 800, `#0058be`, blue-tinted long shadows, glass on grid, Material Symbols) but reinterprets composition, rhythm and moments of delight.

## What's new vs. the existing site
- **Composition:** floating glass pill nav replaces the flat bar; the hero collapses headline + proof into a single asymmetric 7/5 split with a live product mock.
- **Rhythm:** stronger sectional cadence — kickers, oversized numerals, intentional whitespace.
- **Editorial moments:** a "BYOK constellation" dark panel, a "credit ledger" bento tile, a pricing strip with true price-per-task math, an agentic trace timeline.
- **Density:** each section is a single idea, sized to breathe.

## Files
- `index.html` — clickable demo (home → features → pricing toggle → agent demo)
- `Nav.jsx`, `Hero.jsx`, `TrustRow.jsx`, `HowItWorks.jsx`, `FeatureBento.jsx`, `AgentTrace.jsx`, `Providers.jsx`, `PricingStrip.jsx`, `FAQ.jsx`, `Footer.jsx`
- `ui.jsx` — atoms (Button, Chip, Kicker, Card, Icon)

## Run
Open `index.html`. No build step.
