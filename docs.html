downloadInstallation

  • Upload the pressark folder to /wp-content/plugins/ or install from the WordPress plugin installer.
  • Activate PressArk from the Plugins menu.
  • Requirements: WordPress 6.0+ and PHP 8.0+.
  • After activation, the PressArk menu appears in the admin sidebar.
  • Supports single sites and multisite with per-site activation only.
wp plugin activate pressark

tuneConfiguration

  • Onboarding wizard guides initial setup.
  • Choose Managed Billing (PressArk credits via Freemius) or BYOK.
  • Configure Site Profile: brand, audience, style, and tone for generation context.
  • Select post types for Content Indexing.
  • Set retention windows: action logs (default 90 days), chat history (180), cost ledger (365), configurable down to 7 days.

keyBYOK

  • BYOK means Bring Your Own Key.
  • Supported providers: OpenRouter, OpenAI (GPT-4), Anthropic (Claude), DeepSeek, Google Gemini.
  • Freemius license is still required for plugin access.
  • In BYOK, content goes directly to your selected provider (not routed through PressArk servers).
  • API keys are encrypted at rest using Sodium authenticated encryption.
  • BYOK removes credit budget constraints; provider limits/rate limits still apply.

paymentsPricing & Credits

Free ($0, 100K credits, 1 site, DeepSeek V3), Pro ($19/mo, 5M credits, Claude Sonnet 4), Team ($49/mo, 15M, 5 sites), Agency ($99/mo, 40M, 25 sites), Enterprise ($199/mo, 100M, unlimited).

Credits are universal billing units using ICU character counting for consistent cross-model pricing.

Top-ups: 800K/$5, 2.6M/$15, 6M/$30.

Billing cycle per request: Estimate → Reserve → Settle → Reconcile.

Cadence & Approval Modes

expand_more
Cadence: minutely, hourly, daily, weekly, monthly, yearly. Approval policies: Editorial (human confirm), Conservative (previews), Permissive (auto-approved).

Limits, Notifications, and Logs

expand_more
Tier limits: Free 0, Pro 5, Team 15, Agency 50, Enterprise unlimited. Telegram notifications include success/failure/policy blocks. Manual “run now” is available, and all actions are logged with undo capability.

widgetsTools Overview

Content: read/create/edit/delete, bulk edit, find & replace, tone-matched generation.

SEO: analyze/fix SEO, Yoast/RankMath/AIOSEO, meta/OG/focus keyword tooling.

WooCommerce: products/orders/customers/refunds/analytics/inventory/shipping/reviews/emails/webhooks (active only with WooCommerce).

Elementor: widget/container editing, template/forms/popup/global styles (active only with Elementor).

Media: list/get/update/delete, thumbnail regeneration, bulk media ops.

Site Config: settings, menus, themes, customizer, plugin management.

Users & Comments: user management, moderation, bulk reply.

Security & Diagnostics: vulnerability scans, performance/crawl/email/query/cache diagnostics.

Database: stats, cleanup, optimization, revision history.

Block Editor: Gutenberg block CRUD, ACF, FSE templates.

Workflows: deterministic multi-step sequences for content/SEO/store ops that use 40–60% fewer credits.

buildTroubleshooting

  • Chat missing: confirm JS loads, test plugin conflicts, and verify user has pressark_manage_settings (Editor+).
  • API errors: verify key validity/expiry and provider status.
  • Credits depleted: review cost ledger or switch to BYOK for unlimited request volume.
  • Automations not running: ensure WP-Cron is enabled, or install Action Scheduler.
  • Multisite: per-site activation is required, network-wide activation is unsupported.
  • Privacy: keys encrypted with Sodium, retention is configurable, WordPress Privacy Export/Erase is supported.
  • Support: hello@pressark.ai or support@pressark.ai.
