Getting Started

Deploy the architectural power of AI directly into your WordPress core. Follow this sequence to initialize PressArk.

1

Install from ZIP

Download the latest release and upload via the WordPress Plugin uploader. Requires PHP 8.0+.

2

Activate the plugin

Navigate to your Plugins dashboard. Click 'Activate' on PressArk to initialize the background workers.

3

Connect Billing or BYOK

Choose managed credits for instant access or bring your own API key for direct provider integration.

4

Open wp-admin Interface

Locate the PressArk menu in your WordPress sidebar. The dashboard provides real-time system diagnostics.

5
